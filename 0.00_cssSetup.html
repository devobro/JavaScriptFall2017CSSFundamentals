Css Notes

The purpose of CSS is

  • Can be included within html elements using the style attribute
  • Css can be used with the style html element
  • CSS is used for altering colors, fonts, backgrounds, etc.
  • CSS can implement some effects and animations

CSS Statements

  1. Declaration: the statement as a whole with curly braces
  2. Selector: the name of the part you want to sytle
  3. Property: properties to edit (e.g., width, height, background, etc.
  4. Value: numbers, colors, urls - values that we assign
